# Date Format Options for Dashboard

**Created:** January 1, 2026  
**Last Updated:** January 1, 2026  
**Status:** Proposal

---

## Current Format

**"3 days ago"** - Relative time format

## Alternative Options

### Option 1: Short Date + Time
**"Dec 29, 8:34 AM"**
- Pros: Shows actual date, compact
- Cons: Less "human" feeling

### Option 2: Month Day Format
**"Dec 29"** or **"December 29"**
- Pros: Very compact, shows date
- Cons: No time, might be confusing if multiple same day

### Option 3: Relative + Date Hybrid
**"3 days ago (Dec 29)"**
- Pros: Best of both worlds
- Cons: Longer

### Option 4: Smart Relative
**"Dec 29"** for > 7 days, **"3 days ago"** for < 7 days
- Pros: Context-aware
- Cons: Inconsistent format

### Option 5: ISO Short
**"2025-12-29"**
- Pros: Unambiguous, sortable
- Cons: Less readable

---

## Recommendation

**Option 1: "Dec 29, 8:34 AM"** - Clean, shows date and time, easy to scan
