## The Simple Path

### For New Agents

**Step 1:** Read [GETTING_STARTED_SIMPLE.md](docs/guides/GETTING_STARTED_SIMPLE.md)

**Step 2:** Use 3 tools:
- `onboard()` ‚Äî Get identity
- `process_agent_update()` ‚Äî Log work
- `get_governance_metrics()` ‚Äî Check state

**Step 3:** That's it. Explore more when curious.

### For CLI Users

**Use the simple wrapper:**
```bash
python3 scripts/unitares_lite.py onboard
python3 scripts/unitares_lite.py update "What you did" complexity=0.5
python3 scripts/unitares_lite.py metrics
```
